git的理解

发表于 2020-04-29 更新于 2020-04-30 分类于 学习 137 1 分钟

git 它就是一个版本管理工具，方便在于保存文件的不同版本，可以撤回

所有的文件都处于四种状态，未跟踪，已修改，已暂存，已提交

只要触发提交就会产生新的版本。

想要恢复文件

  • 提交前，可以恢复到上一次提交的状态，不管是已修改还是已暂存
  • 提交后，就要回退版本，可以回退到任意一次提交后的状态

1602液晶简单的使用

RS:数据（1）/命令选择端（0）

RW：读（1）/写选择端（0）

E：使能信号

实际上只需要操作以上三个接口

一般液晶用来显示数据，不需要读数据，所以RW位一直置低电平

步骤为，选择命令端，送命令，使能信号由高到低，数据选择端，送数据，使能信号由高到低

注意51单片机中数据连接P0口
